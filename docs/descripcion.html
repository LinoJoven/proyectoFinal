DESCRIPCION

Herramientas y Tecnologías Clave

El desarrollo de este sitio web y el análisis presentado se ha apoyado en un conjunto de herramientas y tecnologías modernas que han facilitado cada etapa del proceso, desde la escritura del código hasta la visualización interactiva de los datos.

Visual Studio Code (VS Code)

Visual Studio Code es un editor de código fuente avanzado, desarrollado por Microsoft, que se ha convertido en un estándar en la industria del desarrollo web. Para este proyecto, VS Code ha sido el entorno principal donde se ha escrito, organizado y depurado todo el código HTML para la estructura de las páginas, CSS para definir los estilos visuales y la presentación, y JavaScript para la interactividad, especialmente en la implementación de los mapas. Sus características, como el resaltado inteligente de sintaxis, las sugerencias de código, la terminal integrada y el robusto sistema de extensiones, han optimizado considerablemente el flujo de trabajo.

Leaflet.js

Leaflet.js es una potente biblioteca de JavaScript de código abierto, ligera y flexible, diseñada específicamente para crear mapas interactivos y amigables con dispositivos móviles. En este proyecto, Leaflet ha sido fundamental para la representación geográfica de la información. Se ha utilizado para desplegar mapas base detallados, posicionar marcadores que indican ubicaciones de interés (como los puntos donde habitan los animales) y dibujar polígonos que delimitan áreas específicas (como la zona de estudio del impacto del ruido en el Parque Simón Bolívar). Su facilidad de uso y su amplia documentación han permitido integrar funcionalidades cartográficas complejas de manera eficiente.

Asistente de IA Gemini

Gemini, un modelo de lenguaje grande y multimodal desarrollado por Google AI, ha desempeñado un rol de asistente virtual a lo largo de la creación de este proyecto. Su participación ha sido crucial para: generar y refinar fragmentos de código en HTML, CSS y JavaScript; proponer soluciones a desafíos de programación y diseño; ayudar en la estructuración y redacción de los contenidos textuales (como esta misma descripción); y ofrecer explicaciones sobre diversos conceptos técnicos. La colaboración con Gemini ha permitido agilizar el desarrollo, explorar diferentes enfoques y asegurar una mayor calidad en el resultado final del sitio web.
